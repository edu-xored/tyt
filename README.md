# tyt

Track your time

## How to build and run

Inside project root:

* `go build`
* `tyt`

## How to init database

Inside project root:

* run tyt
* `cd scripts` 
* `npm install`
* `node init.js`

## Watch dev mode

* `go get -u github.com/githubnemo/CompileDaemon`
* `./run.sh`
